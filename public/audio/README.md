# Login ambient audio

Place your MP3 file here:

```
frontend/public/audio/login-ambient.mp3
```

The login screen loads this file at low volume (`/audio/login-ambient.mp3`).

**Tips**
- Use a short, loop-friendly ambient track (30–90 seconds)
- Keep file size under ~500 KB when possible
- If the file is missing, a soft Web Audio fallback plays after the first tap on the login form
